# Web Skills Quiz App
A responsive and interactive quiz application created using HTML, CSS, and JavaScript.  
This project tests basic web development knowledge through multiple-choice questions with a futuristic UI design.

## Features
- Dynamic question loading
- Interactive option selection
- Previous and Next navigation
- Final score calculation
- Restart quiz option
- Modern glassmorphism-inspired interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
